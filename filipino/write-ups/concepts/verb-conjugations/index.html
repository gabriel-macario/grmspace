Verb Conjugations

In Filipino (at least in FIL002) there are 4 main verb conjugations (all based off of a given root word):

  • Infinitive (Neutral)
  • Past Tense (Perpektibo)
  • Present Tense (Imperpektibo)
  • Future Tense (Kontemplatibo)

Note: The infinitive form of a verb is also used to give a command (e.g. "You, sit!").

All verbs also have some sort of prefix, infix, or suffix. The rules for conjugating a verb (or for making a root word into a verb) are dependent on what prefix/infix/suffix is being used.

Note: Not every prefix/infix/suffix can be used for every root word; knowing which root words work with which prefix/infix/suffix is a matter of knowledge and experience.

Close 'Introduction'

The rules for um- verbs are fairly straight forward and (like all verbs) are built off the root word:

-Um- Verbs Rules (Consonant Root Example)
Rule Example
Root N/A takbo
Infinitive um before first vowel of root tumakbo
Past um before first vowel of root tumakbo
Present first two syllables of infinitive + root tumatakbo
Future first syllable of root + root tatakbo

It is worth noting that with -um- verbs, both the infinitive/command and the past conjugation are exactly the same, so the only way to tell which conjugation is being used in a sentence is through context.

The conjugation rules for -um- remain the same regardless of whether or not the root word begins with a vowel:

-Um- Verbs Rules (Vowel Root Example)
Rule Example
Root N/A inom
Infinitive um before first vowel of root uminom
Past um before first vowel of root uminom
Present first two syllables of infinitive + root umiinom
Future first syllable of root + root iinom

As seen, there is nothing particularly complicated about -um- verbs; the pattern is uniform throughout all root words. Just remember that when a Filipino root word starts with a vowel, the first syllable of the root word (which is repeated in the future tense of -um- conjugations) is made up of only that vowel (e.g. iinom or uupo).

Close '-Um- Verbs'

The ma- prefix can denote the ability/capability to do something. For example, when combined with the root word sulat (write), we arrive at masulat (can write).

Note: This is not always the case for verbs with the ma- prefix. Like many other things, knowing which ma- verbs denote the ability to do something is a matter of knowledge and experience.

The conjugation rules for ma- verbs fairly straight forward:

Ma- Verbs Rules (Consonant Root Example)
Rule Example
Root N/A ligo
Infinitive ma + root maligo
Past na + root naligo
Present na + first syllable of root + root naliligo
Future ma + first syllable of root + root maliligo

The rules for ma- conjugations remain the same even if the root word starts with a vowel:

Ma- Verbs Rules (Vowel Root Example)
Rule Example
Root N/A iwan
Infinitive ma + root maiwan
Past na + root naiwan
Present na + first syllable of root + root naiiwan
Future ma + first syllable of root + root maiiwan

Notice how the the infitive and the past conjugations are the same except for the first letter (m and n respectively). Similarly, the present and the future conjugations are also the same except for the first letter (n and m respectively). This m/n/n/m pattern can be found in several other verb prefixes that start with m. Just remember that the present and future have that repeated syllable.

Note: If you're having a hard time figuring out what syllable repeats, syllables (at least in the context of verb conjugations) usually end in a vowel.

The mag- prefix follows the same m/n/n/m pattern of the ma- prefix, though there is a slight difference depending on whether or not the root word starts with a vowel. For the sake of simplicity, we'll start with a root word that starts with a consonant:

Mag- Verbs Rules (Consonant Root)
Rule Example
Root N/A linis
Infinitive mag + root maglinis
Past nag + root naglinis
Present nag + first syllable of root + root naglilinis
Future mag + first syllable of root + root maglilinis

When the root word of a mag- verb starts with a vowel, then a '-' (dash) must be added after the 'mag' (or 'nag'):

Mag- Verbs Rules (Vowel Root)
Rule Example
Root N/A aral
Infinitive mag- + root mag-aral
Past nag- + root nag-aral
Present nag- + first syllable of root + root nag-aaral
Future mag- + first syllable of root + root mag-aaral
Close 'Ma- & Mag- Verbs'

It helps to think of the mang/man/mam- verbs as a family of verbs, with the mang- prefix serving as the parent while man/mam- prefixes are the children that come out to play sometimes.

The mang- prefix should be used whenever the root word starts with any letter except for 'd', 'l', 'r', 's', 't', 'b', or 'p'. In a majority of cases, when the root word starts with a consonant that is not k, mang- verbs follow the m/n/n/m pattern:

Mang- Verbs Rules (Consonant Not 'K')
Rule Example
Root N/A gamot
Infinitive mang + root manggamot
Past nang + root nanggamot
Present nang + first syllable of root + root nanggagamot
Future mang + first syllable of root + root manggagamot

When the root word for a mang- verb does start with a 'k', the mang- prefix assimilates the 'k'. This means that the last letter of the prefix (remember that 'ng' is one letter in Filipino), devours the 'k' and takes it spot, creating what can be thought of as an 'assimilated root.' This slightly alters the rules of conjugation:

Mang- Verbs Rules (Assimilated 'K')
Rule Example
Root N/A kuha
Assimilated Root last letter of prefix + root word without first letter nguha
Infinitive ma + assimilated root manguha
Past na + assimilated root nanguha
Present na + first syllable of assimilated root + assimilated root nangunguha
Future ma + first syllable of assimilated root + assimilated root mangunguha

Note: According to the instructor, it's not only 'k' that gets assimilated by mang-; however, she could not give any examples off of a non-'k' root getting assimilated off the top of her head.

If you're having difficulty with figuring out the first syllable of the assimilated root, you can instead think of it as the second syllable of the infinitive/past. For example, if the infinitive form is manguha, the syllable to repeat is 'ngu', meaning that the present tense is nangunguha and that the future tense is mangunguha.

Fortunately, vowels are never assimilated by mang-. Unfortunately, the rules for root words that start with vowels aren't very straight forward.

Occasionally, a '-' (dash) before root words that start with a vowel: I know that's not much to go on, but there are a few rules of thumb.

Root words that start with an 'a' usually have a dash (mangako being the one exception that the instructor could provide), and root words that start with an 'i' or a 'u' usually don't have a dash. The instructor could not provide examples of what occurs when the root word starts with an 'e' or 'o'.

When the root word for a mang verb starts with a vowel and requires a dash, the conjugation rules follow the m/n/n/m pattern:

Mang- Verbs Rules (Vowel with Dash)
Rule Example
Root N/A agaw
Infinitive mang- + root mang-agaw
Past nang- + root nang-agaw
Present nang- + first syllable of root + root nang-aagaw
Future mang- + first syllable of root + root mang-aagaw

When the root word for a mang verb starts with a vowel and does not require a dash, the conjugation rules follow a slightly altered m/n/n/m pattern:

Mang- Verbs Rules (Vowel Root, No Dash)
Rule Example
Root N/A isda
Infinitive mang + root mangisda
Past nang + root nangisda
Present nang + first letter of root with -ng added + root nangingisda
Future mang + first letter of root with -ng added + root mangingisda

Now, for reasons that may or may not be sensible, when the root word starts with a 'd', 'l', 'r', 's', or 't', the mang- prefix turns into the man- prefix. Root words that begin with 'd', 's', or 't', are commonly assimilated by the man- prefix. In those cases, the conjugations follow the same rules as when mang- assimilates a 'k' root word, except it's the man- that does the assimilation. This pattern can be thought of as the m/n/n/m pattern with assimilation For example:

Man- Verbs Rules (Assimilated Root)
Rule Example
Root N/A dukot
Assimilated Root last letter of prefix + root word without first letter nukot
Infinitive ma + assimilated root manukot
Past na + assimilated root nanukot
Present na + first syllable of assimilated root + assimilated root nanunukot
Future ma + first syllable of assimilated root + assimilated root manunukot

When the man- prefix does not assimilate the root word (usually the case when the root word starts with 'l'), then it follows the m/n/n/m pattern:

Man- Verbs Rules (No Assimilation)
Rule Example
Root N/A ligaw
Infinitive man + root manligaw
Past nan + root nanligaw
Present nan + first syllable of root + root nanliligaw
Future man + first syllable of root + root manliligaw

Note: The instructor could not provide us of any examples of man- verbs whose root words start with 'r'.

The mang- prefix turns into the mam- prefix when the root word starts with a 'b' or 'p'. We finally catch a break with the mam- prefix because so far, in every example given, the mam- always assimilates the 'b' or 'p', so it follows the m/n/n/m pattern with assimilation:

Mam- Verbs Rules
Rule Example
Root N/A pitas
Assimilated Root last letter of prefix + root word without first letter mitas
Infinitive ma + assimilated root mamitas
Past na + assimilated root namitas
Present na + first syllable of assimilated root + assimilated root namimitas
Future ma + first syllable of assimilated root + assimilated root mamimitas

Here's a table of when and how to use the mang/man/mam- prefixes:

Mang/Man/Mam- Usages
Prefix When Root Word Starts With Assimilates Pattern (No Assimilation) Pattern (With Assimilation)
Mam- 'b', 'p' 'b', 'p' N/A (always assimilates) m/n/n/m pattern with assimilation
Man- 'd', 'l', 's', 'r', 't' 'd', 's', 't' m/n/n/m pattern no assimilation m/n/n/m pattern with assimilation
Mang- 'All other letters 'k' Varies (1, 2, 3) m/n/n/m pattern with assimilation
Close 'Mang/Man/Mam- Verbs'

There's a lot to keep in mind when conjugating in/hin- verbs, but first we'll start with the general rules:

In/Hin- Verbs Rules (Consonant Ender Example)
Rule Example
Root N/A gupit
Infinitive root + (h)in gupitin
Past Place in before first vowel of root ginupit
Present first two syllables of past conjugation + root ginugupit
Future first syllable of root + infinitive gugupitin

In most cases (not always), when the root word ends in a vowel, you append a 'hin' to the end of the word instead of an 'in'.

Note: Regardless of whether or not you append an 'in' or a 'hin' to get the infinitive and future conjugations, you will always insert an 'in' for the past conjugation.

Here's an example of the general rules applied to a root with a vowel ender:

In/Hin- Verbs Rules (Vowel Ender Example)
Rule Example
Root N/A kopya
Infinitive root + (h)in kopyahin
Past Place in before first vowel of root kinopya
Present first two syllables of past conjugation + root kinokopya
Future first syllable of root + infinitive kokopyahin

So, generally, there is only a little difference in the rules when a root ends in a vowel. However, there are a few root words that do not fall under the general case. In those exceptions, there are minor tweaks that occur when conjugating for the infinitive, which affects the future tense for in/hin- conjugations (past and present conjugation rules remain are the same across all roots):

  1. Root words whose last vowel is an 'o' -> Turn the 'o' to a 'u' and then add the appropriate suffix (usually 'in').
  2. Irregular Roots -> Drop the last vowel from the root and then add the appropriate suffix ('in' or 'hin').

Here are some examples illustrating the above tweaks:

In/Hin- Examples (Exceptions)
Root Infinitive Past Present Future
putol (a) putulin pinutol pinuputol puputulin
ayos (a) ayusin inayos inaayos aayusin
gamot (a) gamutin ginamot ginagamot gagamutin
dala (b) dalhin dinala dinadala dadalhin
gawa (b) gawin ginawa ginagawa gagawin
kuha (b) * kunin kinuha kinukuha kukunin

* - Kuha is an example of a where Filipino gets kind of wonky sometimes. By regular rules, you'd drop the 'a' from kuha and add 'hin'. But kuhhin looks really weird, especially in Filipino, so it becomes kunin. Please note that this not the actual scientific explanation for whatever is going on with kuha.

There is one more odd thing about in/hin- (and an/han-) verbs: In certain cases (usually when the root word starts with an 'l') after inserting an 'in' into a word (not appending it), it is acceptable to switch the first 'l' and the first 'n' - you know, for magical linguistic reasons. So, lininis can also be written as nilinis and linilinis can be written as nililinis.

Note: 'Luto' would normally give the following conjugations: 'linuto' & linuluto'. Then, following the above, 'linuto' -> 'niluto', and 'linuluto' -> 'niluluto'. However, the one text I used as reference does not list either 'linuto' or 'linuluto' as proper conjugations for 'luto', so I'm not actually sure what goes on with that specific case.

Close 'In/Hin- Verbs'

The conjugation rules for an/han- Verbs are nearly identical to the rules for in/hin- verbs (including the exceptions to the general cases). The only real change is that there is always an -an/han at the end of every conjugation, not just for the infinitive and the future. So, here are the general an/han- rules:

An/Han- Verbs Rules (Consonant Ender Example)
Rule Example
Root N/A tawag
Infinitive root + (h)an tawagan
Past Place in before first vowel of infinitive tinawagan
Present first two syllables of past conjugation + infinitive tinatawagan
Future first syllable of root + infinitive tatawagan

In most cases (not always), when the root word ends in a vowel, you append a 'han' to the end of the word instead of an 'an' (similar to in/hin-).

Note: Regardless of whether or not you append an 'an' or a 'han' to get the infinitive and future conjugations, you will always insert an 'in' for the past conjugation.

Here's an example of the general rules applied to a root with a vowel ender:

An/Han- Verbs Rules (Vowel Ender Example)
Rule Example
Root N/A pinta
Infinitive root + (h)an pintahan
Past Place in before first vowel of infinitive pinintahan
Present first two syllables of past conjugation + infinitive pinipintahan
Future first syllable of root + infinitive pipintahan

So, once again, there is little difference in the conjugation rules regardless of whether or not the root word ends in a vowel. However, there are a few root words that do not fall under the general case. In those exceptions, there are minor tweaks that occur when conjugating for the infinitive, which will affect all tenses of an/han- verbs (unlike with in/hin- verbs):

  1. Root words whose last vowel is an 'o' -> Turn the 'o' to a 'u' and then add the appropriate suffix (usually 'an').
  2. Irregular Roots -> Drop the last vowel from the root and then add the appropriate suffix ('an' or 'han').

Here are some examples illustrating the above tweaks:

An/Han- Examples (Exceptions)
Root Infinitive Past Present Future
ayos (a) ayusan inayusan inaayusan aayusan
tulong (a) tulungan tinulungan tinutulungan tutulungan
sara (b) sarhan sinarhan sinasarhan sasarhan
takip (b) takpan tinakpan tinatakpan tatakpan
laba (b) ** labhan linabhan/nilabhan linalabhan/nilalabhan lalabhan

** - Read below

There is one more odd thing about an/han- (and in/hin-) verbs: In certain cases, usually when the root word starts with an 'l', after inserting an 'in' into a word (not appending it), it is acceptable to switch the first 'l' and the first 'n' - you know, for magical linguistic reasons. So, linabhan can also be written as nilabhan, and linalabhan can be written as nilalabhan.

Close 'An/Han- Verbs'

The conjugation rules for i- verbs are extremely straight forward. There are a number of ways to make state the rules and they will all be correct; For reasons related to the format of the tests, I prefer to structure rules around the infinitive when possible:

I- Verbs Rules (Consonant Example)
Rule Example
Root N/A turo
Infinitive i + root ituro
Past Place in before second vowel of infinitive itinuro
Present first three syllables of past conjugation + root itinuturo
Future Infinitive with repeated second syllable ituturo

I- verbs conjugate nearly the same when the root word starts with a vowel (pay attention to the past conjugation):

I- Verbs Rules (Consonant Example)
Rule Example
Root N/A alis
Infinitive i + root ialis
Past Place ni before second vowel of infinitive inialis
Present first three syllables of past conjugation + root iniaalis
Future Infinitive with repeated second syllable iaalis

Note: There seems to be one exception to the general i- verb rules: when the root word starts with two consonants. In both the instructor's documents and the class's text, the only i- verb root that fulfills this crtiera is 'prito', whose conjugations are: iprito/ipinirito/ipiniprito/ipriprito. Notice that only the past tense conjugation deviates from the normal rules, (just like with vowel-roots) with an ini being placed after the p (similar to vowel-roots). I assume this has something to do with linguistic reasons.
