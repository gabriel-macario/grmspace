Requests (Mga Pakiusap)

The prefix 'paki-' alongside the article nga is used to denote requests, similar to using the English "please". The person being asked to do something is denoted with a ng-marker/pronoun (most commonly mo for a singular person or ninyo for multiple people). The topic/goal of the request is denoted with an ang-marker/pronoun (e.g. ang, ang mga, si, sina, ako, ikaw/ka, siya, tayo, kami, kayo, sila, ito, iyan, iyon).

Goal-Focused Verb + Actor + Topic Goal
Pakiabot (nga) mo ang bulaklak
ninyo (nga)

Note: The 'nga' would go before a 'mo' but would come after a 'ninyo'.

Commands (Mga Utos)

To give a command for in/hin- verbs, you have to use a ng-marker/pronoun to denote who you are telling to do the action. For commands, the most common pronouns will be mo or ninyo. The topic/object/goal will be denoted by an ang-marker/pronoun (e.g. ang, ang mga, si, sina).

Example: Kainin mo ang gulay. ((You) Eat the vegetable.)

Close '3. Opposites'

As with everything ever in life, if you've been keeping up, this shouldn't be an issue. The only new stuff here are the i-, in/hin-, and an/han- verbs.

Because she'll likely give us an example conjugation of the verb, you don't need to worry about any of the wonky stuff: Assimilation, glottal stops, dashes, irregular verbs, etc. There is only one exception to this, and that's with in/hin- verbs. But before that, we'll start with conjugation rules for all the other verb forms.

Note: There are many ways to write out the rules for conjugations; For test-taking purposes, the ones listed below are structure such that given any single conjugation, you can arrive at any other conjugation. If you have already established your own rule system that gives you the correct answers, don't worry about what I've written below. Do what works for you.

-Um- Verb Rules
Rule Example
Root N/A takbo
Infinitive um before first vowel of root tumakbo
Past same as infinitive tumakbo
Present first two syllables of infinitive + root tumatakbo
Future first syllable of root + root tatakbo

Note: Um- verbs conjugate the same even if root word starts with a vowel: 'alis' -> 'umalis/umalis/umaalis/aalis'

.
Ma- Verb Rules
Rule Example
Root N/A ligo
Infinitive ma + root maligo
Past turn first m of infinitive to n naligo
Present past with repeated second syllable naliligo
Future turn first n of present to m maliligo

Ma- verbs conjugate the same even if root word starts with a vowel. In the appropriate chapter of the text, there is a dash, but the instructor told us not to put that dash. Additionally, the ma- verb table in the text's appendices does not have a dash: 'uhaw' -> 'mauhaw/nauhaw/nauuhaw/mauuhaw'.

Also, this m/n/n/m pattern (with the repeating syllable for the present and future conjugations), is mirrored across every verb prefix that starts with an 'm' (at least that we've learned in class): ma-/mag-/mang-/man-/mam-. Take a look:

Mag- Verb Rules (General)
Rule Example
Root N/A linis
Infinitive mag + root maglinis
Past turn first m of infinitive to n naglinis
Present past with repeated second syllable naglilinis
Future turn first n of present to m maglilinis

If the root word starts with a vowel, you put a dash after the prefix: 'iwan' -> 'mag-iwan/nag-iwan/nag-iiwan/mag-iiwan. But, assuming an example is provided on the test, it should be obvious whether or not a dash is needed.

Mang/Man/Mam- Verb Rules
Rule Example
Root N/A gamot
Infinitive mang/man/mam + (assimilated) root manggamot
Past turn first m of infinitive to n nanggamot
Present past with repeated second syllable nanggagamot
Future turn first n of present to m manggagamot

Note: Sometimes, when the root word starts with a vowel, mang- verbs will have a dash after the prefix. Again, whether or not a dash should be included will be evident if the instructor gives example conjugations.

These three prefixes are lumped together because these are the only 'm' prefixes where assimilation occurs. But if you don't know what that is, or when it happens, or whatever, you don't have to worry about it. The assimilation, if there is any, will have already occurred in any example she gives, so you just need to apply/reverse the rules to get the necessary conjugation. The examples below illustrate this:

  • (mang- verb w/ assim.) 'kuha' -> 'manguha/nanguha/nangunguha/mangunguha'
  • (man-- verb w/ assim.) 'dukot' -> 'manukot/nanukot/nanunukot/manunukot'
  • (man-- verb w/o assim.) 'ligaw' -> 'manligaw/nanligaw/nanliligaw/manliligaw'
  • (mam-- verb w/ assim.) 'pitas' -> 'mamitas/namitas/namimitas/mamimitas'

The only thing that might trip you up is the repeated syllable. Generally, if the last letter of the prefix is next to a vowel, then the syllable to repeat made up of that last letter of the prefix + the vowel that follows. (remember that 'ng' is one letter in Filipino). Otherwise, it should be obvious what syllable to repeat. Refer to the above to see this in effect.

On to an/han- verbs:

An/Han- Verb Rules (General)
Rule Example
Root N/A tawag
Infinitive root + (h)an tawagan
Past Place in before first vowel of infinitive tinawagan
Present first two syllables of past conjugation + infinitive tinatawagan
Future infinitive with repeated first syllable tatawagan

Whether you place a 'an' or 'han' is dependent on a number of factors (irregular root, vowel placement, probably other stuff). Once again, 'an' vs 'han' will be addressed by the example given (so will any shenanigans like 'o' -> 'u' or droping the last vowel from the root), and thus, the above rules will hold for any an/han- verb (when working from a given conjugation). See below:

  • (irreg. verb) 'takip' -> 'takpan/tinakpan/tinatakpan/tatakpan
  • (irreg. verb) 'bukas' -> 'buksan/binuksan/binabuksan/bubuksan
  • (irreg. verb) 'sara' -> 'sarhan/sinarhan/sinasarhan/sasarhan
  • ('o' -> 'u' verb) 'ayos' -> 'ayusan/inayusan/inaayusan/aayusan
  • ('o' -> 'u' verb) 'tulog' -> 'tulungan/tinulungan/tinutulungan/tutulungan

The rules for i- verbs are super duper similar to the rules for an/han- verbs, you just have to account for the added 'i' (which changes up some of the counting). Hopefully the similarity is easily recognizable:

I- Verb Rules
Rule Example
Root N/A sulat
Infinitive i + root isulat
Past place in before second vowel of infinitive isinulat
Present first three syllables of past conjugation + infinitive isinusulat
Future infinitive with repeated second syllable isusulat

With an/han/i/in/hin- verbs, whenever there is an 'in' inserted into a word (which occurs in the past and present conjugations) whose root began with an 'l', it is acceptable to switch the first 'l' with the first 'n'. So for 'luto', 'ilinuto/iniluto' are acceptable past tense conjugations, and 'ilinuluto/iniluluto' are acceptable present tense conjugations. Notice that the above rules for conjugation still work, regardless of whether or not the 'l'/'n' switch occurs.

Note 1: In the back of the book, for the in-form of 'luto', 'niluto' and niluluto' are the only conjugations given for the past and present. I do not know whether or not that means 'linuto' and 'linuluto' are incorrect.

Note 2: Also from the back of the book, the root word 'prito' varies slightly from the rules given above for i-verbs (specifically the past-tense): 'prito' -> 'iprito/ipinirito (notice that it's 'ini' and not 'in')/ipiniprito/ipriprito'. I do not know why this is and I cannot (easily) find any other examples of this. If I had to guess, it has to do with the double-consonant beginning.

By now, hopefully it's clear that as long as an example conjugation is given, the above rules can be used to get you to any other conjugation. So there is no need to worry about assimilation, glottal stops, figuring out whether or not there's a dash, or whatever randomness that's ever-present in Filipino (at least with those verb-infixes).

However, a similarly 'sturdy' set of rules (centered around a given conjugation) cannot be created for in/hin- verbs. This is because of: (a) the existence of irregular in/hin- verbs (like with an/han- verbs), and (b) not all in/hin- conjugations can be arrived at from the infinitive/other conjugations (unlike with an/han- verbs).

Uh, it's easier to see when you take a look at the general in/hin- verb rules:

In/Hin- Verb Rules (General)
Rule Example
Root N/A gising
Infinitive (modified) root + (h)in gisingin
Past Place in before first vowel of unmodified root ginisng
Present first two syllables of past conjugation + unmodified root ginigising
Future infinitive with repeated first syllable gigisingin

So, unlike with an/han- verbs, in/hin- verbs do not remain consistently modified across all conjugations.

Take the root 'ayos for example: (in/hin-) 'ayusin/inayos/inaayos/aayusin' vs. (an/han-) 'ayusan/inayusan/inaayusan/aayusan. Note how the 'o/u changes. Additionally, you cannot necessarily tell if an 'in' is required or if a 'hin' is required - even given an example (e.g. 'inayos' does not tell you if it's 'ayusin or 'ayushin').

Anyway, what all that means is that you have to learn when an in/hin- root needs to be modified, which roots need an 'in' vs a 'hin', and which roots are irregular. Regarding the first two things:

  • The most common modification occurs when the last vowel of a root is an 'o': If it is, then the 'o' turns to a 'u' for the infinitive and future conjugations.
  • More or less, if the root ends in a consant, use 'in'; if it ends with a vowel, use 'hin'. Like with anything else, there are exceptions (that I unfortunately do not know).

As for irregular verbs, our kind professor has given us a list of irregular verbs for just that purpose, so study up.

For convenience, the irregular in/hin- verbs are listed below - sans definitions (remember, the above an/han- rules already take into account irregular verbs):

Time Elements

  • Past: ka- word (kahapon, kanina, kamakalawa), noong phrases (noong Sabado, noong Enero, noong umaga), the word 'noon' itself.
  • Present: 'tuwing', 'ngayon' and ngayon phrases (ngayong umaga, ngayon tanghali), stuff that repeats (gabi-gabi, araw-araw, linggu-linggo)
  • Future: 'bukas', 'mamaya', mamaya phrases (mamayang gabi, mamayang hapon), 'balang araw' (and presumably 'balang linggo' or 'balang buwan'), sa- words/phrases (samakalawa, sa Linggo, sa Enero)

TL;DR

  • The above conjugation rules will get you from any given conjugation to any other given conjugation, except for the in/hin- rules. For those, you'll just have to learn them normally.

First, the basics:

Ng/Sa- Markers
Ng- Markers Sa- Markers
Singular Plural Singular Plural
Common ng ng mga sa sa mga
Proper ni nina kay kina

(Some) Uses of Ng- Markers:

  • Denoting Ownership
    • Common Structure: Owned Item + Ng- Marker + Owner
    • Ex: ang bola ng bata (the child's ball)
  • Denoting Doer (for in/hin-, an/han-, and i- verbs)
    • Common Structure: Verb + Ng- Marker + Doer
    • Ex: Binili ko ang damit. (I bought the dress.)

(Some) Uses of Sa- Markers:

  • Denoting Ownership
    • Common Structure: Sa- Marker + Owner + Noun
    • Ex: Sa mga estudyante ang mga libro iyon. (Those are the students' books.)
  • Denoting Recipient/Purpose/Location of Verb
    • Common Structure: Verb + Denoted Doer (or Recip/Purp/Loc) + (para) Denoted Recip/Purp/Loc (or Doer)
    • Ex: Gumawa ako ng kard para sa babae. (I made the card for the woman.)

      'ni': This didn't fit any of the usual structures for the uses of sa- markers, and it perfectly fit the ownership structure of ng- marker. Vicky is a singular, proper noun, so 'ni'.

  2. Magaganda ang mga kasuotan (costumes) _____ magagaling na mananayaw

      'ng mga': Knowing that 'kasuotan meant costumes helped a lot, since a ng- marker comes after a noun and before a linked adjective (or linked noun) We know that we have to use the plural ng- marker because the adjective linked is pluralized via the repeated syllable (magagaling) and it's a common noun, so ng mga is the correct answer.

Since it's been brought up so much, and is useful to know (both for the test and in general), here's the list of sa- markers:

  • akin - singular, first person
  • iyo - singular, second person
  • kaniya/kanya - singular, third person
  • amin - plural, first person, exclusive
  • atin - plural, first person, inclusive
  • inyo - plural, second person
  • kanila - plural, third person

Note: These are often linked to another verb via an -ng (e.g. 'aking', 'kanyang').

Close '5. Ng- & Sa- Markers'

'Nasaan' is used to ask where something is at. Most often, this means you should use 'nasaan' before nouns and such.

'Saan' is used to ask where an activity is occurring at, where something is being done, or to inquire about the destination. Usually, you should use 'saan' before verbs.
