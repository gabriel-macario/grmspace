There are three marker types: ang markers, ng markers, and sa markers. The marker that should be used mainly depends on three things:

  • Singular vs Plural
  • Common vs Proper
  • Usage

First, here's the actual markers:

Marker Table
Ang Markers Ng Markers Sa Markers
Singular Plural Singular Plural Singular Plural
Common ang ang mga ng ng mga sa sa mga
Proper si sina ni nina kay kina

It's not enough to know the words themselves though, you need to also know the appropriate usages for each marker.

Note: While the below information is correct and should get you the correct answer on the final (at least most of the time), the way it is presented might not be technically proper. Technically, ang markers are ALWAYS denote the subject; it's just that what constitutes the subject changes based on the verb affix being used. For example, um- verbs are actor focused verbs: Therefore, in sentences with um verbs, the actor/doer is the subject of the sentence and is marked with an ang marker. Contrast this with -in verbs, which are goal/object-focused verbs. This means the subject of the sentence is the goal/object - NOT the doer - and therefore, the goal/object of the sentence receives the ang marker - NOT the doer. If you're confused by this (and if you trust me), don't worry too much. The usages presented below simply take the necessary logical conclusions from the rules above and lumps them together for ease of learning. I'm mostly just putting this note here because I don't like leaving information out.

Common Usages of Ang Markers

  • Denotes Subject of Sentence
  • Denoting Doer of Um / Mag / Ma / Mang / Man / Mam Verbs
    • Common Structure: Verb + Ang Marker + Doer(s)
    • Examples:
      1. Tumakbo ang bata. (The child ran.)
      2. Mag-aawit ang mga estudyante (The students will study.)
      3. Natutulog si Felice. (Felice is sleeping).
      4. Nangingisda sina Rod at Earnest. (Rod and Earnest are fishing.)
  • Denotes Goal/Recipient of In/Hin / An/Han / I Verbs
    • Common Structure: Verb + Ng Marker + Doer + Ang Marker + Direct Object
    • Examples:
      1. Niluto ng kusinero ang adobo (The chef cooked the adobo.)
      2. Nililinis ng diyanitor ang mga mesa. (The janitor cleaned the tables.)
      3. Ibinigay ng lalaki si Jer ng mga bulaklak. (The man gave Jer flowers.)
      4. Lulutuhan ni Gabe sina Ned at Maria . (Drake will cook for Ned and Maria.)
  • Identificational Sentences
    • Identificational sentences identify one person as another.
    • Common Structure: Ang Marker + Noun/Adjective + (ay) + Ang Marker + Noun/Adjective
    • Examples:
      1. Si Trump ang pangulo. (Trump is the president.)
      2. Si Precy ay ang guro. (Precy is the teacher.)
      3. Ang diyanitor ang musikero (The janitor is the musician.)
      4. Sina Lonzo at Brandon ang mga manlalaro (Lonzo and Brandon are the players).

Tips for Ang Markers

Basically, every sentence should have an ang marker, so if you don't see one, really consider using one. Also, most sentences should only have one ang marker, so if you do already see one, strongly consider using another marker type. The exceptions to these are identificational sentences (see above) and sentences with parallel/independent clauses (e.g. 'Tumakbo si Lois at kumain si Jaryd' ~ Lois ran and Jaryd ate.)
